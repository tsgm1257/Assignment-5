Question 1: What are the different ways to select an element in the DOM?
There are many ways to select elements in the DOM
  • getElementById()
  • getElementsByClassName()
  • getElementsByTagName()
  • querySelector()
  • querySelectorAll()
Question 2: What is the difference between innerHTML, innerText, and textContent?
The innerHTML returns HTML contents with tags and styling. While the innerText only returns text content with styling without the tags. Finally, textContent returns only the text content without any tags or stylings.
Question 3: What is event delegation in the DOM?
Event delegation is where using an event listener on a parent element to handle events on it's child elements.
Question 4: What is event bubbling in the DOM?
Event bubbling is where an event on an element first triggers event listeners on that element, then on its parent, then grandparent, and so on up the DOM tree
Question-5: How do you create, add, and remove elements using JavaScript?
  1. Create: document.createElement(tagName)
  2. Add: parent.appendChild(child)
  3. Remove: parent.removeChild(child)
